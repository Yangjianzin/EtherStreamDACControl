# EtherStreamDACControl
## 功能:
### (1)與PC透過Ethernet進行通訊資料交換，與指令通訊。

#### (A)接收傳輸資料
#### (B)讀取目前傳送資料資訊:已傳輸筆數

#### (C)啟動資料串流 跟 結束資料串流功能

### (2)透過Ethernet接收資料:30,000筆資料，能在3秒內傳完。

## 使用硬體
### Arch Max V1.1 x1
### MCP4725 x2

## 使用技術

### C++
### Mbed OS
### TCP/IP
